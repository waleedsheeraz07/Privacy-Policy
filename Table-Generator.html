Privacy Policy

Table Generator  •  Effective Date: 09/02/2026

Table Generator is a simple Android application that generates multiplication tables (from 1 to 10) based on a number entered by the user. This Privacy Policy explains how the app handles user information.

Table Generator does not collect, store, share, or transmit any personal or non-personal data.

Information Collection and Use

The app does not require user registration and does not request any personal information. The number entered by the user is processed only to generate the multiplication table and is immediately discarded.

  • No data collection of any kind
  • No local or remote storage
  • No background data processing
  • No ads, analytics, or tracking technologies

Offline Functionality

Table Generator operates entirely offline. An internet connection is not required, and the app does not perform any network communication.

Third-Party Services

This application does not integrate with any third-party services, SDKs, advertising platforms, or analytics tools.

Children’s Privacy

Table Generator is suitable for users of all ages. Since the app does not collect any data, it fully complies with children’s privacy protection regulations.

Data Security

As no data is collected or stored, there is no risk of unauthorized access, data leakage, or misuse.

Changes to This Privacy Policy

This Privacy Policy may be updated if the app’s functionality changes. Any updates will be published on this page.
